# AI Bus Project 🚍

An intelligent bus route and travel assistance system built using Python and Flask. This project helps users find bus routes, timings, stops, travel duration, and transportation details efficiently through a simple web-based interface.

## 📌 Features

* 🚌 Smart bus route search
* 📍 Route details with bus stops
* ⏱ Estimated travel duration
* 💰 Approximate ticket pricing
* 🔄 Frequency and timing information
* 🌐 Flask-based web application
* 📊 Clean and user-friendly interface
* ⚡ Fast and lightweight backend system

## 🛠 Technologies Used

* Python
* Flask
* HTML/CSS
* JavaScript
* API Integration
* Git & GitHub

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/AI-Bus-Project.git
```

### 2️⃣ Open Project Folder

```bash
cd AI-Bus-Project
```

### 3️⃣ Install Required Packages

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python backend.py
```

### 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
